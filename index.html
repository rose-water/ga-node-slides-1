What is it?
  • +Node is a JS Runtime built with Chrome's V8 Javascript Engine
  • +V8 compiles js code to machine code, making it really fast
  • +So far we've been running javascript in the browser
  • +With Node, we can use JS to run on our machines as standalone apps
  • +We will use Node to run server-side javascript

Express
Creating our web server - index.js
  • +On the first line we use the require keyword to essentially import the express module
  • +We assign it to a const variable called express - it's an alias, can be anything
  • +In the second line we create an instance of an express application
  • +REFERENCE: express() 
  •                   
                    const express = require('express');
                    const app     = express();
Express
Creating our web server - index.js
  • +We define a port number for our app to listen for connections
  • +When we call app.listen() we pass in the port # & a callback function 
  •                   
                    const express = require('express');
                    const app     = express();
                    const PORT    = 3000;
                    
                    app.listen(PORT, function() {
                      console.log('listening on PORT: ' + PORT);
                    });
