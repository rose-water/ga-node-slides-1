Intro to HTTP
HyperText Transfer Protocol
  • +This is the way that clients and servers communicate with each other
  • +Servers are just a bunch of machines that are running in a remote location
Intro to HTTP
Basic Flow
  • 1.A client (a browser) sends an HTTP request (e.g., get some data, update existing data)
  • 2.A web server receives the request
  • 3.The server runs an application (in our case, Node) to process the request
  • 4.The server returns an HTTP response (output) to the browser (we are doing this today)
  • 5.The client (the browser) receives the response
Intro to HTTP
Types of Requests
  • +There are a number of different types of http requests that you can make
  • +Some of the big ones: GET, PUT, POST, DELETE
  • +A GET request is for retrieving data only. No changes to data stored server side.
  • +A POST request is for sending data to a server (e.g. form submissions)

Express
Serving static files
  • +REFERENCE: Express docs: Serving static files
  • +Create a folder called public
  • +Inside the public folder, create an index.html
  • +Set up your index.html structure
  • +Inside of index.js, add/change the following: 
  •                   
                    /* tell our app where to serve our static files */
                    app.use(express.static('public'));
                    
                    /* define a route */
                    app.get('/', function(req, res) {
                      /* send a static file */
                      res.sendFile(__dirname + '/index.html');
                    });
Express
Defining another endpoint: request query
  • +let's define another endpoint called /test
  • +In your browser: http://localhost:3000/test?species=cats
  • +the ? in this url refers to the start of a 'query string'
  • +A query string is made up of key-value pairs
  • +This query string has one pair: key = species, value = cats 
  •                   
                    app.get('/test', function(req, res) {
                      let cats = ["Scottish Fold", "Russian Blue", "Ragdoll"];
                      let dogs = ["Golden Retriever", "German Shepherd", "Alaskan Malamute"];
                      
                      if (req.query.species == 'cats') {
                        res.send(cats);
                      } else if (req.query.species == 'dogs') {
                        res.send(dogs);
                      } else {
                        res.send("Unknown Species");
                      }
                    });
